# CRYPTO PRICE CHECKER

## PROJECT STRUCTURE:
app.py              # Main Flask application  
templates/          # HTML pages (index + results)  
static/style.css    # Basic styling  
requirements.txt    # Python dependencies  

## HOW TO RUN:
pip install -r requirements.txt  
python3 app.py  

## GO TO SAFARI AND GO TO:
http://127.0.0.1:5000  

## API THAT WAS USED:
https://api.coinbase.com/v2/prices/{crypto}-{currency}/spot  

## PURPOSE OF THIS PROJECT:
- Flask routing  
- HTML form handling  
- Using APIs in Flask  
- JSON parsing  
- Displaying processed data in templates
